# learning-claude-code

A small quote generator web app.

## Features

- **Random quotes** — click **New quote** to display a new quote.
- **Copy to clipboard** — click **Copy quote** to copy the current quote (with author) to your clipboard.
- **Dark/light theme toggle** — the button in the top-right corner switches themes. Your choice is remembered and the app respects your system preference on first load.

## Running

Open `index.html` in any modern browser. No build step or dependencies required.
